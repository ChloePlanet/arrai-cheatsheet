Awesome Arr.ai CheatSheet

  • Transform

    • transforms a single value

      ->

      @> "hello, " -> . ++ "world"

      hello, world

    • transforms the members of a set

      =>

      @> {2, 4, 6, 8, 10} => . + 1

      {3, 5, 7, 9, 11}

    • transforms the entries of a sequence or dictionary

      >>

      @> [1, 2, 3, 4] >> 2 ^ .

      [2, 4, 8, 16]

      @> {"A": 10, "B": 42, "C": 100} >> 100 - .

      {'A': 90, 'B': 58, 'C': 0}

    • transforms the entries of a sequence or dictionary

      >>>

    • transforms the attributes of a tuple

      :>

      @> (r: 0.5, g: 0.2, b: 0.7) :> 1 - .

      (b: 0.3, g: 0.8, r: 0.5)

  • Tuple

    (a: 1, b: 2)

    • Merge

      +>

      @> (a: 1, b: 2) +> (b: 3, c: 4)

      (a: 1, b: 3, c: 4)

      @> (a: 1, b: (c: 2)) +> (b: (c: 4), c: 4)

      (a: 1, b: (c: 4), c: 4)

      @> (a: (b: 1)) +> (a: (c: 2))

      (a: (c: 2))

  • Dictionary

    {"a": 1, "b": 2} / {(@: "a", @value: 1), (@: "b", @value: 2)}

    • Merge

      +>

      @> {"a": 1, "b": 2} +> {"b": 3, "d": 4}

      {'a': 1, 'b': 3, 'd': 4}

      @> {"a": {"b": 1}} +> {"a": {"c": 2}}

      {'a': {'c': 2}}

  • Set

    {1,2,3,4}

    • Filter

      where / filter

      @> {(a:1, b:2), (a:2, b:3), (a:2, b:4)} where .a=2

      {(a:2, b:4)}

      @> {(a:1, b:2), (a:2, b:3), (a:2, b:4)} where .a=2 and .b=3

      {(a:2, b:3)}

      @> {(a:1, b:2), (a:2, b:3), (a:2, b:4)} where .a=2 or .b=4

      {(a:2, b:3), (a:2, b:4)}

      @> {(a:1, b:2), (a:2, b:3), (a:2, b:4)} where .a!=2

      {(a:1, b:2)}

      @> {1, [2, 3], 4, [5, 6]} filter . {[a, b]: a + b}

      {5, 11}

      @> {1, [2, 3], 4, [5, 6], [7, 8, 9]} filter . {[a, ..., b]: a + b}

      {5, 11, 16}

      @> {1, [2, 3], 4, [5, 6]} filter . {[_, _]: 42}

      {42}

    • Evaluate each item

      =>

      @> {1,2,3,4} => .^2

      {1,4,9,16}

  • Array

    [1,2,3] / {(@: 0, @item: 1), (@: 1, @item: 2), (@: 2, @item: 3)}

    • Evaluate each item

      >>

      @> [1,2,3,4] >> .^2

      [1,4,9,16]

  • Relations

    {|a,b| (1,2), (2,2)}

    • Join

      <&>

      @> {|a,b| (1,2), (2,2)} <&> {|a,c| (1,2), (1,3)}

      {(a: 1, b: 2, c: 2), (a: 1, b: 2, c: 3)}

    • Nest

      nest |attr1,attr2,...|key

      @> {|a,b| (1,2), (2,2)} <&> {|a,c| (1,2), (1,3)}

      {(a: 1, b: 2, c: 2), (a: 1, b: 2, c: 3)}

      @> {|a,b| (1,2), (2,3)} nest |b|nestb

      {(a: 1, nestb:{(b: 2)}), (a: 2, nestb:{(b: 3)})}

      @> {|a,b,c| (1,2,3), (1,3,3), (1,2,2)} nest |b|nestb

      {(a: 1, c: 3, nestb: {(b: 2), (b: 3)}), (a: 1, c: 2, nestb: {(b: 2)})}

      @> {|a,b,c| (1,2,3), (1,3,3), (1,2,2)} nest |b,c|nestbc

      {(a: 1, nestbc: {(b: 2, c: 3), (b: 3, c: 3), (b: 2, c: 2)})}
